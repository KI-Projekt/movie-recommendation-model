API Beschreibung

Die Moviemate API ist speziell für Kinos entwickelt und hilft dabei, den Besuchern personalisierte Filmauswahlen zu präsentieren. Basierend auf den zuvor eingegebenen Daten und Präferenzen der Benutzer, schlägt die API Filme vor, die dem individuellen Geschmack der Besucher entsprechen. Diese API kann nahtlos in Kino-Websites integriert werden und verbessert das Benutzererlebnis durch personalisierte Empfehlungen.

Diese API ist vollkommen kostenlos

POST /api/data
Host: moviemate.mabu2807.de
Content-Type: application/json
  • Es ist werder Cors noch eine Authenifizierung notwendig

Beispie Request-Body

{ "movies": [ { "externalId": "", "title": "Muster Titel", "year": 2021 } ], "user_ratings": [ { "externalId": "" "title": "Movie Title", "year": 2021, "rating": 5 } ] }

Bespiel Response

{ "movies": [ { "externalId": "1", "score": 19, "title": "Test", "year": 2011 }, { "externalId": "2", "score": 18, "title": "Test2", "year": 2012 } ] }

Hinweise

  • Die Daten der User müssen selbst erhoben werden!
  • Um so mehr Daten Übermittelt werden, umso genauer ist die Vohersage!
  • Das mitgegebenen User-Rating zu einem Film sollte auf einer Skala von 0-5 sein. 0 = schlecht, 5 = sehr gut
  • Die zurückgegebenen Filme sind auf einer Skala von 0 bis 100 bewertet.
  • Bei der externalId können sie ihre eigene ID aus der Datenbank mitgeben. Diese ist ausschließlich dazu da, damit Sie die Filme Scorings nach erhalten der Response Ihren Filmen zuordnen können.
